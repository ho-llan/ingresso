# Ingresso
### Terceiro projeto do curso de JavaScript
A finalidade do projeto é simular a compra de ingressos de um evento.

Funções do projeto:

 - Comprar ingresso: <br>
  -Cadeira Inferior <br>
  -Cadeira Superior <br>
  -Pista <br>
 - Escolher a quantidade de ingressos. <br>
 
 
 A partir disto verificar se a quantidade está disponivel ou não e abater na quantidade de estoque.
